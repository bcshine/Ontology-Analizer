# 중간계 온톨로지 관계망분석기

온톨로지 기반의 관계망을 분석하고 시각화하는 도구입니다.

## 주요 특징***ㅇㅇㅇ

- 🎨 순수 HTML/CSS로 구현된 애니메이션 네트워크 로고
- 📱 완벽한 반응형 디자인 (모바일, 태블릿, 데스크톱)
- ✨ 인터랙티브 UI 효과
- 🚀 빠른 로딩 속도 (외부 이미지 없음)

## 배포

이 프로젝트는 GitHub Pages를 통해 배포됩니다.

자세한 배포 방법은 `DEPLOYMENT_GUIDE.md`를 참고하세요.

## 사용 방법

웹사이트에 접속하여 "분석하기" 버튼을 클릭하면 Gemini 기반 분석 도구로 이동합니다.

## 기술 스택

- HTML5
- CSS3 (애니메이션, 그라데이션, SVG)
- 반응형 웹 디자인
- GitHub Pages

## 로컬 테스트

`index.html` 파일을 브라우저에서 직접 열어 테스트할 수 있습니다.

## 지원 브라우저

- Chrome (권장)
- Firefox
- Safari
- Edge
